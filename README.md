# 太阳能数据系统 (Solar Data System)

## 项目简介

这是一个太阳能数据管理和分析系统，用于收集、存储和分析太阳能设备的运行数据。

## 项目结构

```
solardatasystempy/
├── data/                    # 数据文件目录
│   ├── 19.99V 6.00A data_detail_1_2025-05-09T12-15-19.xlsx
│   ├── 20.2V 19.8Ω 1.3Adata_detail_1_2025-05-02T06-23-00.xlsx
│   └── 39.9V 9.02A data_detail_1_2025-05-09T13-02-36.xlsx
├── image/                   # 图片资源目录
│   ├── 11-MeterSphere 测试质量大屏.png
│   ├── 20230201172443396.png
│   └── banner-2x.webp
└── README.md
```

## 数据文件说明

- `19.99V 6.00A data_detail_1_2025-05-09T12-15-19.xlsx` - 19.99V电压，6.00A电流的测试数据
- `20.2V 19.8Ω 1.3Adata_detail_1_2025-05-02T06-23-00.xlsx` - 20.2V电压，19.8Ω电阻，1.3A电流的测试数据  
- `39.9V 9.02A data_detail_1_2025-05-09T13-02-36.xlsx` - 39.9V电压，9.02A电流的测试数据

## 功能特性

- 太阳能设备数据收集
- 数据存储和管理
- 数据分析和可视化
- 测试质量监控

## 技术栈

- Excel数据处理
- 数据可视化
